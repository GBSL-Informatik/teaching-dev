# Teaching Dev

Die Lernplattform Teaching Dev basiert auf einer [statisch generierten Webseite](https://github.com/GBSL-Informatik/teaching-dev) und einer [API](https://github.com/GBSL-Informatik/teaching-api) zum Abspeichern von Daten. Beide Komponenten können auf einem Dokku-Server gehostet werden - sofern die Webseite aber nicht generell passwortgeschützt ist, empfiehlt sich ein schnellerer statischer Webhoster (Github Pages, Cloudflare Pages, Netlify).
