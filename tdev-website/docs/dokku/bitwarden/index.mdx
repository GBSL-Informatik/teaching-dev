# Bitwarden

[Vaultwarden](https://github.com/dani-garcia/vaultwarden#installation) ist eine inoffizielle Implementierung des Bitwarden-Passwortmanagers, die in Rust geschrieben ist. Sie bietet eine leichtgewichtige und effiziente Alternative zum offiziellen Bitwarden-Server und ist vollständig kompatibel mit den offiziellen Bitwarden-Clients.

<DynamicInput name="app" default='bitwarden' monospace />
<DynamicInput name="domain" default='bitwarden.domain.ch' monospace />


<TemplateCode>
    ```bash title="root@dokku"
    # create app
    dokku apps:create {{app}}

    # setup and mount persitent storage
    mkdir -p /var/lib/dokku/data/storage/{{app}}/data
    dokku storage:mount {{app}} /var/lib/dokku/data/storage/{{app}}/data:/data

    # add port mapping
    dokku ports:set {{app}} http:80:80
    # set domain
    dokku domains:set {{app}} {{domain}}
    
    # use X-Forwarded-For header to resolve users IP
    dokku config:set --no-restart {{app}} IP_HEADER="X-Forwarded-For"

    # enable admin route
    dokku config:set --no-restart {{app}} ADMIN_TOKEN=$(openssl rand -base64 48)

    # for correct propagation of the clients ip:
    dokku nginx:set {{app}} x-forwarded-for-value "\$proxy_add_x_forwarded_for"


    nano /home/dokku/{{app}}/ENV
    ```
</TemplateCode>
